display:inline

Rule 1: Requires box-sizing: border-box to use percents (e.g. 3 columns with width 20%/50%/30%)

Rule 2: No spaces between column DIVs

means ...</div><div..., space between DIVs would insert space betweens blocks on the page, so you can't set width in percents (e.g. left DIV width: 30% and right DIV width: 70%)

Rule 3: requires style vertical-align: top

float:left

Rule 1: Requires box-sizing: border-box to use percents (e.g. 3 columns with width 20%/50%/30%)

Rule 2: requires DIV with style clear: both after all DIVs with float:left

Usually empty DIV with class class="clearfix" is placed right after float DIVs and class "clearfix" is defined in CSS like .clearfix { clear: both; }
